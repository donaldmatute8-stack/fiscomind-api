## 💡 RAZONAMIENTO FISCAL PARA FISCOMIND

### Principio de clasificación de CFDIs recibidos:

Al analizar CFDIs recibidos (efecto "I"), el sistema debe clasificar:

1. **¿Quién emitió la factura?**
   - Si es BANCO (BBVA, Banorte) → Probablemente es ingreso personal/reembolso
   - Si es TELECOM (AT&T, Telcel) → Probablemente gasto personal
   - Si es TRANSPORTE, SERVICIOS → Posiblemente gasto de negocio

2. **¿Cuál es el monto?**
   - Montos pequeños recurrentes → Probablemente suscripciones personales
   - Montos grandes únicos → Investigar origen

3. **¿Cuál es el uso CFDI?**
   - Si dice "G03" (gastos) → Puede ser deducible
   - Si dice "P01" (por definir) → Clasificar según contexto

### Lógica de minimización legal:

1. **Maximizar deducciones legales:**
   - Solo gastos con CFDI válido a nombre del contribuyente
   - Solo gastos relacionados con la actividad económica
   - Solo gastos con documentación de soporte

2. **Facilidades de pago cuando hay escasez:**
   - Presentar a tiempo aunque sea incompleto
   - Usar programas de condonación si aplican
   - Nunca dejar de presentar (multa = 20-100% del omitido)

3. **Prioridad de urgencia:**
   - Primero: Regularizar periodos anteriores
   - Segundo: Presentar meses actuales a tiempo
   - Tercero: Minimizar montos con deducciones reales
